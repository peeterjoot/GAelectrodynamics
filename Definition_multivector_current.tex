\makedefinition{Multivector current.}{dfn:isotropicMaxwells:660}{
The \textit{current} ([\si{V/m^2}] (Volts/square meter)) is defined as
\begin{equation*}
J = \eta \lr{ c \rho - \BJ } + I \lr{ c \rho_\txtm - \BM }.
\end{equation*}
} % definition
